Inspection Panel

See what's in your model. Diagnose what's hidden. Catch fab errors before the shop floor.

🔄
Sync Views
Match the pan, zoom, and rotation of the active view to one or more target views. Works across 2D plans (crop regions and zoom) and 3D views (camera position and orientation). Keeps coordination views aligned during review.
👁️
Element Visibility
Pick an element and two views — the tool walks every visibility condition (category, workset, filter, phase, design option, crop region, view range) and reports exactly which setting causes the difference. Stop manually toggling settings to find why something disappeared.
🔬
Snoop Selection
Deep-inspect any element's full property tree: parameters, geometry, connectors, fabrication data, and raw .NET members. Essential for debugging API behavior, verifying parameter values, and understanding how Revit stores element data.
⚖️
Compare
Side-by-side comparison of two elements, families, views, or schedules. Highlights every parameter difference in a sortable grid. Verify consistency between similar elements or track changes between model versions.
🔎
Find Sheet
Select a view or viewport and instantly navigate to the sheet where it lives. Saves time hunting through the Project Browser on large models with hundreds of sheets.
📐
Check Duct Orientation
Scans fabrication duct and flags any piece where the width dimension is narrower than the depth. Catches orientation errors before they reach the shop floor. Color-codes affected elements in the active view.
🌀
Coil Duct
Calculates overall cut length (body + connector allowances) for each rectangular fab straight and checks coil-machine eligibility. Tags and color-codes results: green = eligible, red = bad connector/shape, purple = exceeds coil width. Configurable connector profiles with import/export.

Designation Panel

Number it. Tag it. Convert it. Spin it. Sync the hangers.

🔢
Renumber
Wizard-driven renumbering with multiple ordering modes: left-to-right, top-to-bottom, nearest-neighbor, or along a path. Supports prefix/suffix patterns, grouping by system or level, and gap numbering. Preview mode shows proposed numbers before committing. Undo-safe.
🏷️
Place Tags
Batch-place tags on selected elements using saved tag profiles. Supports offset positioning, leader control, and multi-category tagging in a single pass. Profile system remembers your preferred tag family and settings per category.
🪟
View Converter
Converts Legend views to Drafting Detail views and back. Rebuilds detail curves, text notes, filled regions, annotation symbols, and dimensions directly in the new view. Options to delete the source, auto-open the result, and zoom to fit.
🔠
Toggle Case
Cycle TextNote elements between UPPERCASE, Title Case, and lowercase — like Shift+F3 in Word. Auto-Cycle mode detects the dominant case across your selection and advances to the next. Save profiles for your preferred mode.
🔄
Rotate Multiple
Rotates each selected element around its own center point — unlike Revit's native rotate, which uses a single shared axis. Enter an angle and every selected element spins in place. Built for adjusting diffusers, equipment, or fittings in bulk.
🔁
Convert
Unified conversion engine for fabrication ductwork. Convert between fabrication and native Revit elements, change service types, or swap connector configurations. Handles rectangular, round, and oval profiles with automatic size mapping.
🪝
Set Hangers
Pre-select duct or pipe (or pick after launching). Finds all Generic Model RFA hangers within 1 ft XY of each element and writes slope-interpolated BOH elevation, clamp size, trapeze width, and insulation parameters. Pick Host lets you manually reassign a hanger.
